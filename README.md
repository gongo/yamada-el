## Usage

```lisp
(require 'yamada)

(global-set-key (kbd "C-c C-c C-y") 'yamada-dancing)

(yamada-dancing 1) ;; or C-c C-c C-y (M-x yamada-dancing)
(yamada-dancing 5) ;; or C-u 5 C-c C-c C-y
```
